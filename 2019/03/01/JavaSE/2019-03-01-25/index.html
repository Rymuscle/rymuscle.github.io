25. 数组常用操作 (ArrayUtils)

2019-03-01
  • JavaSE

java.util.Arrays

提供了大量的静态方法用于对数组进行一些常见操作 (有空指针问题)

  1. public static <T> List<T> asList(T... a) : 返回由指定数组支持的固定大小的列表 (转换成的列表, 不支持 add、remove 操作)

  2. public static String toString(Object[] a) : 将数组用字符串形式展示 {1,2,3} 结果为 [1,2,3]

  3. sort(float[] a)/sort(float[] a, int fromIndex, int toIndex) : 将数组升序排序 (也可将数组中的指定范围进行排序)

  4. sort(T[] a, Comparator<? super T> c) 、sort(T[] a, int fromIndex, int toIndex, Comparator<? super T> c)
