BeanUtils工具类

2019-02-10
  • JavaWeb

概述

  1. BeanUtils 工具类是用于封装JavaBean的, 可以用来简化数据的封装

  2. JavaBean (标准的Java类) 的基本规范：

    • 类必须被public修饰
    • 必须提供空参的构造器
    • 成员变量必须使用private修饰
    • 需要提供公共 setter 和 getter 方法

  3. 成员变量 和 属性

    • 成员变量: 类种定义的变量即成员变量
    • 属性: setter, getter 方法 去掉 get, set 后, 剩余的名字首字母转小写后, 即是属性
    • 一般成员变量名和属性是一样的, 但是也可以使用 setHaHa() 去给成员变量 name 赋值, 此时成员变量name和属性名haHa就不一样了

  4. 使用, 需要引入 commons-beanutils-1.9.3.jar, commons-logging-1.2.jar 包 (https://mvnrepository.com/artifact/commons-beanutils/commons-beanutils)

方法

  1. setProperty() // 操作的是属性, 可不是成员变量名
  2. getProperty()
  3. populate(Object obj , Map map): 将map集合的键值对信息, 封装到对应的 JavaBean 对象中
