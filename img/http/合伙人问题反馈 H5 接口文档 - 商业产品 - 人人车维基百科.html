    接口1：新增问题

    URL:  /api/v1/questionFeedback/createQuestion

    HTTP：POST

    Content-Type: application/json

    参数
    参数类型
    是否必填
    含义
    备注
    titleString是标题 
    contentString是内容描述 
    catalog1String是一级分类 
    catalog2String否一级分类 
    catalog3String否一级分类 
    attachmentString否附件地址 
    workOrderIdString是工单id 


    响应结果

    返回值示例
    ?
    1
    2
    3
    4
    {
        "status": 0,
        "err_msg": "ok"
    }
