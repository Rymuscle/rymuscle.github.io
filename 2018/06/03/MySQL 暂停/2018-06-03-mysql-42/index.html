42. 高性能索引 -- 独立的列

2018-06-03
  • MySQL

本篇非常简单

  1. 独立的列 是指索引列不能是表达式的一部分, 也不能是函数的参数;

  2. 示例: 很容易看出 where 中的表达式其实等价于 actor_id=4, 但MySQL却无法自动解析这个方程式

    (就像 mysql的查询优化器不能将 id>5 和 id>6 这两个查询条件优化合并成一个 id>6 一样, MySQL优化器有时候没有我们想的那么自然, 但的确非常有用)
