1. IntelliJ IDEA

2018-11-11
  • IDEA

基础快捷键

  1. Ctrl+Alt+L : 代码格式化
  2. Alt+Enter : 导入包
  3. ctrl + 点击方法 从调用处, 跳到方法处, Ctrl + Alt + <- 再调跳回 方法的调用处
  4. Ctrl+d : 删除当前行
  5. Alt+Shift+ 上,下键 : 移动本行代码
  6. 批量修改变量名: 选中变量名, shift+F6, 写新的变量名即可

跳转

  1. 最近打开的文件 Command + E

    项目之间的跳转

    当IDEA 项目窗口比较多的时候, 可以使用这种方式
  2. 上一个窗口：Command+shift+ `
  3. 下一个窗口：Command+ `

文件之间的跳转

Ctrl + e 调出最近打开的文件(同时还可以进行搜索筛选)

最近修改文件切换

Ctrl + Shift + E

代码便捷操作

  1. psvm : 生成主函数
  2. sout : 打印
  3. 5.fori : for (int i = 0; i < 5; i++) { }
  4. Ctrl+Enter : 生成 getter、setter、构造方法、toString 等方法

常用设置

  1. 修改类头部的文档信息: Editor -> File and Code Templates -> File Header, 可以设置如下

  2. 列编辑模式: Alt+鼠标左键 即可进入列编辑模式

  5. 忽略大小写的提示 (一般也没有人去设置)

tips

  1. 初次运行程序, 如果报错 Error:java: 无效的源发行版: 9 (解决方案: https://www.cnblogs.com/dongling/p/8670852.html)

IDEA

  1. 右键代码文件, 可以通过 “Show in Explorer” 查看字节码文件 (貌似在Mac上 没看到, 可以通过 “File”->”Project Structure” 找到 “out” 编译目录)
  2. 查看编译目录, 内部类的.class字节码文件, 命名格式为 外部类名$内部类名.class
