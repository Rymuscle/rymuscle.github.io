2018-12-03
  • JavaSE

main 主函数

  1. 格式是固定的, 被jvm所识别和调用

    • public : 因为权限必须是最大的;
    • static: 不需要对象的, 直接用主函数所属类名调用即可;
    • void: 主函数没有具体的返回值;
    • main: 函数名, 不是关键字, 只是jvm识别的固定的名字;
    • String[] args: 这是主函数的参数列表, 是一个数组类型的参数, 而且元素都是字符串类型;

  2. 虚拟机在运行的时候, 可以给主函数传值, 只不过一般情况下不传

  3. main 不光是main 方法, 还可以当做普通方法

基本语法

  1. 逻辑运算只有 &形式与&&(短路)形式 (没有php中的and, or, 一般也只用短路符)

  2. switch语句: switch(变量) : switch语句的变量的数据类型只能是 byte, short, int, char, 枚举, String
