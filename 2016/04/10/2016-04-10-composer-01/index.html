01.安装

2016-04-10
  • Composer

composer.phar文件下载

先下载composer.phar文件, 3种方式:

  • curl -sS https://getcomposer.org/installer | php
  • php -r "readfile('https://getcomposer.org/installer');" | php
  • 手动下载Composer, 地址

Linux/Unix/OS

  1. 全局安装 : mv composer.phar /usr/local/bin/composer

  2. 局部安装 : mv composer.phar /局部目录/composer

windows

  1. 全局安装 :

    • 配置composer.phar文件路径(D:\WWW\composer)到环境变量中 ;

    • 接下来需要在composer.phar同级目录下新建文件composer.bat :
      echo @php "%~dp0composer.phar" %*>composer.bat

    • 这就安装好了

  2. 局部安装 :

    • 直接把下载的composer.phar文件放到项目根目录下;

    • 运行命令 : php composer.phar install

    • 然后就安装成功了(每个局部目录都需要这么安装, 比较麻烦)

推荐全局安装即可.

Composer是PHP中的一个依赖管理工具, 它可以让你声明自己项目所依赖的库，然后它将会在项目中为你安装这些库。安装步骤可以到官网阅读Getting Started。在继续阅读之前，请确认composer已经安装并且可以使用。命令别名是composer，所以后续的所有如composer xxx均代表是执行composer命令。
